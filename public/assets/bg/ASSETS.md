# Background / Parallax Layers

All images must be **1920 × 1080 px**, PNG, seamlessly tileable horizontally.

| File              | Layer | Parallax | Description                                         |
|-------------------|-------|----------|-----------------------------------------------------|
| `sky.png`         | 0     | 0.10×    | Amber/orange sky, clouds, distant sun               |
| `city.png`        | 1     | 0.30×    | Ruined city skyline silhouettes, mountains          |
| `midground.png`   | 2     | 0.60×    | Destroyed structures, broken power lines            |
| `foreground.png`  | 3     | 1.00×    | Active terrain edge, rocks, debris on ground level  |

## Art Style Reference
Matching `image_1ba2ae.jpg`: desaturated ochre/brown wasteland palette,
heavy atmospheric haze, high-contrast silhouettes, post-apocalyptic feel.

## Recommended Tools
- Aseprite or Photoshop for pixel-art tiles
- Midjourney/Stable Diffusion for concept → manual clean-up
- Keep file sizes < 300 KB each (PNG compress with pngquant)
